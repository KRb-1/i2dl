# i2dl
Introduction to deep learning

## Repository Contents

| Exercise | Topic | Description |
|---|---|---|
| `exercise_01` | Organization and Setup | Sets up the Python environment, Jupyter Notebook workflow, and exercise submission system. It also provides a short introduction to the structure of the course exercises. |
| `exercise_03` | Datasets and DataLoaders | Implements data preprocessing, dataset handling, and mini-batch loading. The exercise introduces the abstractions required to efficiently provide training and validation data to a model. |
| `exercise_04` | Solver and Linear Regression | Implements linear regression and the corresponding optimization pipeline. It covers loss computation, gradient-based optimization, and the structure of a reusable solver. |
| `exercise_05` | Neural Networks | Builds fundamental neural-network components from scratch using NumPy, including layers, activation functions, losses, forward propagation, and backpropagation. |
| `exercise_06` | Hyperparameter Tuning | Explores how learning rates, model architectures, regularization, and other hyperparameters affect model performance. Different configurations are evaluated systematically to improve the validation results. |
| `exercise_07` | Introduction to PyTorch | Introduces PyTorch tensors, automatic differentiation, neural-network modules, optimizers, and standard model-training loops. |
| `exercise_08` | Autoencoders | Implements and trains an autoencoder for image reconstruction. The exercise also investigates compressed latent representations learned from the input data. |
| `exercise_09` | Convolutional Neural Networks | Designs and trains a convolutional neural network for facial keypoint detection, with a focus on CNN architecture, regularization, and optimization. |
| `exercise_10` | Semantic Segmentation | Implements a semantic segmentation network for pixel-wise image classification. The exercise covers segmentation architectures, transfer learning, training, and evaluation. |
| `exercise_11` | Transformers | Implements the main components of a Transformer, including token embeddings, positional encoding, scaled dot-product attention, and multi-head attention. |
